プライバシーポリシー

「Links Portal Plus」（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、データの保護に細心の注意を払っています。本ポリシーでは、情報の取り扱いについて説明します。

1. データの収集と保存

本拡張機能は、ユーザーの氏名、メールアドレス、パスワード等の個人情報を一切収集しません。ユーザーが登録したリンク、ウィンドウの配置、設定情報は、ブラウザの chrome.storage API を使用して、ユーザーの端末内にのみ保存されます。[cite: 1]

2. 権限の使用目的

本拡張機能が要求する各権限は、機能提供の目的にのみ使用されます。[cite: 1]

  • storage: リンクデータや設定のローカル保存。[cite: 1]
  • alarms: Gmail等の未読数をバックグラウンドで定期確認するため。[cite: 1]
  • scripting / tabs: 右クリック制限の解除機能を動作させるため。[cite: 1]
  • contextMenus: 右クリックからリンクを追加するため。[cite: 1]

3. Gmail および Google Chat のデータ

Gmailの未読数取得は、ブラウザ内のAtomフィードを通じて行われ、数値のみがリンクバッジに反映されます。取得したデータが外部のサーバーや第三者に送信されることはありません。[cite: 1]

4. 第三者への提供

本拡張機能が、ユーザーの利用データを販売、共有、または広告目的で第三者に提供することはありません。

5. お問い合わせ

本ポリシーまたは本拡張機能に関するご質問は、デベロッパーダッシュボードに記載の連絡先までお寄せください。
